# Print group sequential CRT design

Print group sequential CRT design

## Usage

``` r
gs_print_crt(x)
```

## Arguments

- x:

  Object of class `gs_design_crt` created by
  [`gs_design_crt()`](https://leejtding.github.io/gsDesignCRT/reference/gs_design_crt.md).

## Value

Printed output of design parameters and table of stopping boundaries
