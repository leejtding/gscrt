# Plot group sequential CRT design

Plot group sequential CRT design

## Usage

``` r
gs_plot_crt(x)
```

## Arguments

- x:

  Object of class `gs_design_crt` created by
  [`gs_design_crt()`](https://leejtding.github.io/gsDesignCRT/reference/gs_design_crt.md).

## Value

Plot of stopping boundaries for the group sequential CRT design
